# Luxury Design System: Estética Premium

**Estética**: "Dark Mode Minimalism" (Inspiración: Apple, Linear, Porsche).

## Reglas de Composición
- **Espacio Negativo**: Aumentar todos los márgenes (`padding`, `gap`, `margin`) en un **20%** respecto a los valores estándar. El lujo reside en el espacio para respirar.
- **Jerarquía Visual**: Uso claro de contrastes para guiar la mirada.

## Elementos Visuales
- **Glassmorphism Fino**:
  - Bordes sutiles: `border-white/10`.
  - Fondos con desenfoque: `backdrop-blur-md`.
  - Color de fondo: Fondos oscuros profundos (`bg-neutral-950`).
- **Micro-interacciones**: Transiciones suaves y sutiles en estados hover.

## Tipografía Premium
- **Fuentes**: Uso de familias 'Sans' geométricas (Inter, Geist o Manrope).
- **Tracking**: Aplicar `tracking-tight` (espaciado reducido entre letras) específicamente para títulos grandes para un look más moderno y sofisticado.
